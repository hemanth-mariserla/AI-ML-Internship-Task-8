# Task 8: Clustering with K-Means

## Objective
Perform **unsupervised learning** with K-Means clustering to group customers based on their spending behavior.


## Tools & Libraries
- Python 3.x
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Results
-Fitting K-Means model with optimal K = 5
-Silhouette Score: 0.5547
